How to Install and Run

If you are using Linux/UNIX; Git Bash or Shell you need to insert './' in front of each command. In Windows you open the command prompt (cmd) and run the following commands:

  • Open up the Shell/Command prompt and navigate using cd [the location] to the place where the TicTacToe repsoitory was put. The location could for example be C:\TicTacToe\
  • Run the command: gradlew build . Then Gradle will run all the test cases and check if the program is ready; it informs you if there are any fails.
  • Run the command: gradlew run . This command will run the TicTacToe game and a greeting will appear to welcome you in the console. From there you can put in your first mark in the game against your opponent.
